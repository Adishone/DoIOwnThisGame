How to obtain Steam Api Key?

First step to get your own Steam API credentials is to have an Steam Account. To do that you need to access this page https://store.steampowered.com/join, and fill the requested information. You will need to agree with the Steam Api Terms of Use, and that you should have spend more than 5.00 USD in the Steam Store to be able to get the API key. After login go to https://steamcommunity.com/login/home/?goto=devapikey fill your Steam Domain Name, confirm the Steam Api Terms of Use check box and click on the “Register” button. If the registration was correct you should see your new Steam Api Key.
