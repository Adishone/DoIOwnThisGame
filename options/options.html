What is GoG Galaxy and how do I get it?

GOG Galaxy 2.0 is a wonderful concept of a PC games launcher that puts all your favourite games in one place. It's essentially a single library for all your games from Steam, Origin, UPlay and Epic Store, as well as games from GOG.com (naturally). Various integrations in GOG Galaxy 2.0 mean it works with 20 gaming platforms currently. To get it go to GoG Galaxy Site

Where do I find GoG Galaxy Database?

If you are using MAC, default database location will be /Users/Shared/GOG.com/Galaxy/Storage/galaxy-2.0.db
If you are using Windows, default database location will be C:\ProgramData\GOG.com\Galaxy\storage\galaxy-2.0.db
ProgramData is hidden folder, so you can have problems with finding it on your disk.

How to get CSV file with my games?

In Playnite you would need addon to export your games list to CSV: https://playnite.link/addons.html#LibraryExporter_54bf64c6-c453-4cbc-92f8-4960b56f930e Uncheck all fields when exporting as only names are needed.
